# Sumato Frontend

## Overview
Sumato Frontend is a web application providing user authentication, dashboard, product listings, admin management, and contact solutions. The project is structured for easy navigation and modular development.

## Features
- User registration and login
- Admin login and dashboard
- Product listing and management
- Contact form
- Solutions page
- Responsive design

## Getting Started
1. Clone the repository:
   ```bash
   git clone <repo-url>
   ```
2. Open the project in your preferred editor.
3. Serve the files using a local server (e.g., Live Server extension or Python's SimpleHTTPServer).

## Usage
- Open `index.html` for the home page.
- Use `login.html` and `register.html` for authentication.
- Admins use `admin-login.html` and `dashboard.html`.
- Product information is in `products.html`.
- Contact via `contact.html`.
- Solutions are detailed in `solutions.html`.

## Customization
- Update CSS files in `css/` for styling.
- Modify JS files in `js/` for functionality.
- Add icons/images in `assets/icons/`.

## Scripts
- `update-csp.sh`: Script for updating Content Security Policy.
